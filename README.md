# Kaouwu

[![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT/)

An app for browsing, creating, and saving kaomojis on your phone.

<p align="center">
<img src="https://raw.githubusercontent.com/Alexdaz/Kaouwu/main/Assets/SSKaouwu.png" width="340" height="737"/>
</p>

## Features

- Browse curated kaomoji categories by emotion.
- Create custom kaomojis by combining decorations, arms (with emojis), and faces.
- Save custom kaomojis locally and mark favorites.
- Export saved or favorite kaomojis as `.txt`.
- Light, dark, and system theme support.

## Documentation

- See `docs/project_documentation.md` for project structure, data sources, and native icon/splash references.

## Contributing

Contributions are welcome.

- To add more kaomojis, edit `assets/data/kaomojis.json` and follow the existing category structure.
- Feel free to commit JSON updates to add new kaomojis. I can test changes on my side, but it helps a lot if contributors test locally first and verify new entries render correctly in the UI before committing.
- Any other improvements are also welcome (UI, UX, performance, accessibility, localization, docs, or tests).
